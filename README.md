# Baltic Gods

A premium dark editorial mythology website about Baltic gods and sacred geography in Lithuania and Latvia.

## Features

- **Multi-language support**: English, Lithuanian (LT), and Latvian (LV)
- **Interactive map**: Explore sacred sites across Lithuania and Latvia with Leaflet
- **Comprehensive deity database**: Information on 11 Baltic gods including Perkūnas, Saulė, Laima, and more
- **Search functionality**: Find deities by name, domain, or description
- **Responsive design**: Dark editorial aesthetic with moss green and warm amber highlights
- **Static site**: Built with Next.js and optimized for GitHub Pages

## Technology Stack

- **Framework**: Next.js 16 with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS 4.x
- **Internationalization**: next-intl
- **Maps**: Leaflet & React-Leaflet
- **Content**: MDX for deity information
- **Testing**: Jest & React Testing Library
- **CI/CD**: GitHub Actions

## Pages

- **Home**: Hero section with featured deities and search
- **Pantheon**: Browse all deities with domain filters
- **Map**: Interactive map showing sacred locations
- **Deity Detail**: Comprehensive information about each deity with associated locations
- **Field Report**: Form to submit new sacred site information

## Getting Started

### Prerequisites

- Node.js 20 or later
- npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build

```bash
npm run build
```

The static site will be exported to the `out` directory.

### Testing

```bash
npm test
```

## Deployment

The site is configured for automatic deployment to GitHub Pages via GitHub Actions. When changes are pushed to the `main` branch, the site will be automatically built and deployed.

## Deities Included

- **Perkūnas** - Thunder God, Sky Wielder
- **Saulė** - Goddess of the Sun
- **Laima** - Goddess of Fate and Fortune
- **Žemyna** - Mother Earth
- **Velnias** - God of the Underworld
- **Austėja** - Goddess of Bees
- **Mėnulis** - God of the Moon
- **Dievas** - Sky Father
- **Gabija** - Goddess of Fire
- **Bangpūtys** - God of Sea and Storms
- **Medinė** - Goddess of Forests

## Design Philosophy

The website follows a dark editorial aesthetic inspired by museum archives and sacred folklore:

- Black/charcoal backgrounds for depth
- Moss green overlays representing ancient forests
- Warm amber/gold highlights evoking sacred fires
- Huge serif typography for gravitas
- Italic poetic subtitles for mystical atmosphere
- Cinematic forest imagery
- Dark translucent glass-panel cards
- Minimal refined interactions
