# Baltic Gods

A multilingual static website about Baltic mythology, featuring gods, locations, and interactive maps.

## Features

- 🌍 **Multilingual Support**: Available in Lithuanian (LT), Latvian (LV), and English (EN)
- 🏛️ **Pantheon**: Comprehensive database of Baltic gods
- 🗺️ **Interactive Map**: Explore deity locations across the Baltic region
- 📖 **MDX Content**: Rich content stored in MDX format
- 📱 **Responsive Design**: Mobile-friendly interface
- 🎨 **Modern Stack**: Built with Next.js, TypeScript, and React

## Pages

- **Homepage**: Featured gods and sacred locations
- **Pantheon**: Browse all Baltic deities
- **Interactive Map**: Geographic exploration of deity locations
- **Contact**: Report new locations and artifacts
- **God Details**: In-depth information about each deity

## Technology Stack

- **Framework**: Next.js 15+ (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Content**: MDX files
- **Testing**: Jest, React Testing Library, Playwright
- **Deployment**: GitHub Pages

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Run tests
npm test

# Run E2E tests
npm run test:e2e
```

Visit [http://localhost:3000](http://localhost:3000) to see the website.
