## Key Metadata Markers to Seek
- **Region**: Look for "Aukštaitija", "Žemaitija" (Samogitia), "Dzūkija", "Suvalkija", "Kurzeme", "Latgale", "Vidzeme".
- **Tribe**: "Latvians", "Lithuanians", "Old Prussians", "Curonians", "Semigallians", "Jotvingians".
- **Year/Period**: Note early mentions (e.g., 14th century chroniclers like Peter von Dusburg, or 16th century records like Jan Łasicki).
- **References**: Always cite the collector (e.g., Jonas Basanavičius, Krišjānis Barons) or the archive ID.
